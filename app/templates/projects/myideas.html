Project

MyIdeas

Idea management system focused on capturing, editing and evolving ideas, with support for images.

Overview

From tickets to ideas.

MyIdeas is a variation of MyTicketFlow, but centered around ideas instead of tickets. Users can create and manage ideas, update them over time and attach images that help illustrate concepts or designs. There is no priority tag system—only a clean flow to collect and refine ideas.

FastAPI • Python • PostgreSQL • React

The application

The React interface allows users to register ideas, edit them later and associate images to give more context. Instead of ticket fields and priority tags, the focus is on clear descriptions and visual references, making brainstorming and documentation more natural.
