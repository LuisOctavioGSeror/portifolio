Project

MyTicketFlow

Ticket management system with MCP and LLM integration for automated triage and assisted resolution.

Overview

Backend and AI in one stack.

MyTicketFlow is a full‑stack ticket management system. The backend is FastAPI (Python) with PostgreSQL; the frontend is React. MCP and LLMs are used for triage and assisted resolution.

FastAPI • Python • PostgreSQL • React • MCP • LLM

The application

The React UI lets users manage tickets, view status and use AI-assisted actions. Below you can add screenshots of the main screens.

Backend

The API is layered: routers handle HTTP, controllers call services, and services contain the business logic. Persistence is in PostgreSQL via models; request and response shapes are validated with Pydantic schemas.

AI lives in infrastructure: the llm module talks to language models; mcp implements Model Context Protocol. Config and shared setup are in core; utils for helpers. The repo uses Docker and docker-compose for PostgreSQL and local runs.
