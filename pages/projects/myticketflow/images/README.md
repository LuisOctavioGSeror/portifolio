# MyTicketFlow project images

Add your images here. The project page expects:

- **app-1.png** — Screenshot of the app (e.g. main view or dashboard)
- **app-2.png** — Second screenshot (e.g. ticket list or detail)
- **backend-diagram.png** — Backend architecture diagram

You can use other names and update the `src` in `myticketflow.html` if you prefer.
